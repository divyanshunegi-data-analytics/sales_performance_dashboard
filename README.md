# 📊 Sales Performance Dashboard (Power BI)

## 🔍 Overview
This project presents an interactive Sales Performance Dashboard built using Power BI. The goal of this project is to analyze sales data, identify key business trends, and provide actionable insights through data visualization and forecasting.

---

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- Data Cleaning & Transformation
- DAX (Data Analysis Expressions)

---

## 📈 Key Features
- KPI Metrics (Total Sales, Profit, Orders, Average Sales)
- Sales Trend Analysis with Forecasting
- Region-wise Sales Performance
- Product Performance Analysis (Top Products)
- Customer (Sales Rep) Performance Analysis
- Sales Funnel Visualization
- Interactive Filters (Slicers for Region, Category, Date)

---

## 📊 Dashboard Insights
- Sales show a consistent upward trend over time
- A small number of products and customers contribute the majority of revenue
- The West region generates the highest sales
- Consumer segment dominates the sales funnel
- Forecasting indicates continued growth in future sales

---

## 🔮 Forecasting
Forecasting has been implemented using Power BI’s built-in analytics feature to predict future sales trends based on historical data.
